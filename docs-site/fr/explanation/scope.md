# Périmètre

Client TypeScript pour les API JSON NENE2 — pas un second runtime framework.

Hors périmètre : runtime PHP (NENE2), serveur MCP stdio, accès DB direct.

Version stable npm : **`@hideyukimori/nene2-client@1.0.0`**. [CHANGELOG](https://github.com/hideyukiMORI/nene2-js/blob/main/CHANGELOG.md)
