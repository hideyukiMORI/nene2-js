# スコープとミッション

**nene2-js** は NENE2 JSON API を TypeScript で快適・型安全に使うためのクライアントです。第二のフレームワークランタイムにはなりません。

## スコープ内

HTTP クライアント、OpenAPI 整合、Problem Details、npm パッケージ `@hideyukimori/nene2-client`。

## スコープ外

PHP ランタイム（NENE2）、stdio MCP（nene-mcp）、DB 直アクセス（禁止）。

詳細: [GitHub docs/scope.md](https://github.com/hideyukiMORI/nene2-js/blob/main/docs/scope.md)
