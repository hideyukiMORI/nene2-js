# NENE2 との関係

| リポジトリ | 役割                              |
| ---------- | --------------------------------- |
| NENE2      | PHP ランタイム、OpenAPI authoring |
| nene2-js   | TypeScript コンシューマ SDK       |

契約の正は NENE2 の `openapi.yaml`。**OpenAPI が勝つ** — サーバが契約に合えばクライアント修正、違えばサーバまたは契約を先に直す。

[エコシステム](/ja/integrations/ecosystem) も参照。
