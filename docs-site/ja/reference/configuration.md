# 設定

| フィールド | 説明                             |
| ---------- | -------------------------------- |
| `baseUrl`  | API 起点（末尾スラッシュは除去） |
| `apiKey`   | `X-NENE2-API-Key`                |
| `bearer`   | Bearer JWT                       |
| `fetch`    | テスト用注入                     |
| `signal`   | 全リクエストに転送               |

シークレットは SDK が `.env` を読まない — アプリ側の環境変数で渡す。`baseUrl` は信頼できる設定のみ（SSRF 注意）。
